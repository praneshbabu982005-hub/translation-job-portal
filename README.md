# Translation Job Portal

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Folder Structure
- `public/` - Static files
- `src/` - React source code
  - `components/` - Reusable UI components
  - `pages/` - Main pages (Home, About, Contact)
  - `data.js` - Dummy job data
